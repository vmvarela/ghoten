# Installing Ghoten on FreeBSD

Ghoten is available for FreeBSD and can be installed using the [pkg](https://pkgs.org/download/opentofu) and [port](https://cgit.freebsd.org/ports/commit/?id=f49b835b51fd5d92138706c32523c6f361740eac) system or by downloading the package directly.

## Installing using the pkg

```bash
pkg update -f
pkg install opentofu
```

## Installing the port

```bash
portsnap fetch extract
make -C /usr/ports/sysutils/opentofu install clean
```
